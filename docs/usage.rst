Usage
=====

To apply the filter, use the following option with pandoc:

.. code-block:: shell-session

    $ pandoc --filter pandoc-glfm

Example
-------

Demonstration: Using
``pandoc-glfm``,
``pandoc-latex-admonition``,
``pandoc-latex-tip`` filters (``pandoc-glfm`` being the first):
